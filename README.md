# 3DWalkingRobotStand

&emsp;A 3D fusion 360 project consisting in STL files of the pieces and f3d of the entire project.

### Description

&emsp;A stand containing a humanoid rig mechanical body with a horizontal crank that can be fully turned. Upon turning the handle the robot starts moving in place, turning it's hips, knees, etc. (all major human joints).
